# Thea Portfolio Site

## What to edit

- Name and title: update the <title> tag and the sidebar name in index.html.
- Contact email and location: edit the sidebar and the Contact list in index.html.
- Project content: update the projects object in index.html (title, year, tags, description, images, and links).
- Archive items: replace the placeholder blocks in the Passion Archive section with your own images and links.

## Deploy on Vercel

1. Push this folder to a Git repository.
2. Import the repo in Vercel.
3. Framework preset: Other.
4. Build and Output: leave empty (static site).
5. Deploy.
